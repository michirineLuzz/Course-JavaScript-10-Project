# Course-JavaScript-10-Project

## ➤ Deskripsi  
Repo ini berisi 10 proyek kecil yang dibangun sebagai bagian dari kursus “JavaScript 10 Projects in 10 Days”. Setiap proyek dirancang untuk memperkuat pemahaman saya terhadap HTML, CSS, dan JavaScript melalui tantangan nyata dan interaktif.  
Tujuan saya dengan repo ini adalah untuk menunjukkan kemampuan frontend saya mulai dari konsep dasar hingga implementasi yang lebih kompleks.

## ➤ Isi Proyek  
| # | Nama Proyek               | Deskripsi Singkat                                  |
|---|---------------------------|---------------------------------------------------|
| 1 | Love Calculator           | Kalkulator “persentase cinta” sederhana.         |
| 2 | BMI Calculator            | Kalkulator indeks massa tubuh.                    |
| 3 | Loan Calculator           | Menghitung cicilan pinjaman dengan suku bunga.   |
| 4 | Counter Application       | Aplikasi penghitung sederhana (increment/decrement). |
| 5 | Palindrome Checker        | Mengecek apakah kata atau kalimat adalah palindrome. |
| 6 | RGB Color Generator       | Generator warna RGB acak dan penerapan ke elemen UI. |
| 7 | Tip Calculator            | Kalkulator tip otomatis berdasarkan total tagihan. |
| 8 | Time & Date Widget        | Menampilkan tanggal & waktu real-time.            |
| 9 | Unicode Value Viewer      | Memperoleh nilai Unicode dari karakter input.    |
|10| Age Calculator             | Menghitung usia berdasarkan tanggal lahir.        |

> *Catatan: Anda dapat melihat folder masing-masing proyek untuk detail kode dan demo.*

## ➤ Teknologi  
- HTML5 & CSS3  
- JavaScript (Vanilla)  
- Responsive design (mobile-first)  
- Struktur folder tiap proyek siap untuk dikembangkan lebih lanjut  
- Tidak menggunakan library besar agar tetap “lightweight”
